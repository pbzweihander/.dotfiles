# dotfiles

대대손손 물려줄 설정파일들

- `.dotfiles/`
  - `desktops/`: desktop files for `/usr/share/applications/`
  - `docs/`: documents, references, ...
  - `etc/`: config files for `/etc/`
  - `home/`: dotfiles for `~/`
    - `.config/`: config files for `~/.config/`
    - ...
  - `services/`: systemd service files
  - `utils/`: Utilities, scripts, ...
